- 15 free dithering algorithms (Floyd-Steinberg, Atkinson, Jarvis, Stucki, Burkes, Sierra, Bayer, Noise)
- Full Pro algorithm library available with license key
- Real-time preview with background thread processing
- 7 UI themes (Dark, CRT Green, Amber, Game Boy, Pink, Vaporwave, Light)

- Full palette collection (Pro)
- Special imaging effects: Thermal (free), Night Vision, Ultrasound, Oscilloscope, Satellite, Blueprint, Forensic Camera, Redacted Doc, Datamosh (Pro)
- PNG export (free), JPG/SVG/Video export (Pro)
- Batch processing (Pro)
- Copy to clipboard (Pro)
- Pre-processing: brightness, contrast, blur, sharpen, gamma, noise, denoise, invert
- Post-processing: Epsilon Glow, Star Effect, glow threshold + color tint
- Compare view (before/after split slider)
- Presets: save, load, share settings
